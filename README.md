## Web application

Set up your environment:

```shell
virtualenv indel-mapper
source indel-mapper/bin/activate
pip3 install -r requirements.txt
```

Run the application:

```shell
python3 app.py
```

Run the tests:

```shell
python3 -m unittest
```

Deploy to Heroku:

```shell
git push heroku master
```
